# 🧠 AI写作需求智能解析器模块

## 📋 模块概述

**模块名称**：AI-Write-Requirement-Analyzer  
**核心功能**：精准理解和分层解析各类AI写作需求的专业智能解析器  
**技术架构**：NLP语义分析 + 机器学习分类 + 规则引擎 + 历史数据学习  
**服务目标**：将用户的自然语言需求转换为精确的专家匹配参数  
**协作模块**：@.cursor/rules/ai-write-1.0/engine/ai-write-engine-专家智能调度引擎.md、@.cursor/rules/ai-write-1.0/engine/ai-write-engine-角色组合优化器.md

```
智能平台识别系统提示词：

你是一个专业的平台识别专家，需要根据用户的写作需求准确识别目标平台。

【微信公众号识别特征】
直接关键词：公众号、微信公众号、公号、WeChat、微信文章
```
